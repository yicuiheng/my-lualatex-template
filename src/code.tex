    title=中間値の定理]
  区間 $[\alpha, \beta]$ で連続な関数 $f(x)$ について，$f(\alpha)$ と $f(\beta)$ の間にある任意の実数 $c$ に対して，
  ある実数 $k \in (\alpha, \beta)$ を $f(k) = c$ を満たすようにとることができる
\end{tcolorbox}
