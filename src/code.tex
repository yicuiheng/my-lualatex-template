\section{プログラミング関係}

\par プログラムのフォントには \code{inconsolata} を使っています．
\par インラインコードの例:  \verb|\code{hoge + fuga = piyo}| で \code{hoge + fuga = piyo} と表示されます.
\par 複数行にまたがるコードの例:

\begin{program}[linenos]{c}{sample}
  func main() {
      let a = 12
      let b = 30
      a + b
    }
\end{program}
